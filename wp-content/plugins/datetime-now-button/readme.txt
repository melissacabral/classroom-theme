Adds a Now button to the right of date and time fields when editing a Post, Page or Comment. 

== Description ==

The Now Button simply changes the date and time of a post, page, or comment to... now. This useful if you make significant changes to a post, page, or comment and want to push it back to the front page or something of that nature.

Now button will be displayed:

* When editing the date and time on a Post

* When editing the date and time on a Page

* When editing the date and time on a Comment

* When quick editing a Post

* When quick editing a Page

== Installation ==

1. Upload the 'datetime-now-button' directory to the '/wp-content/plugins/' directory
2. Activate 'Date/Time Now Button' from the site's 'Plugins' menu
